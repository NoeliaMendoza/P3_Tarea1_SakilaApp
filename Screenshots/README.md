# Capturas de Pantalla

Agrega aquí las capturas de pantalla de la aplicación funcionando.

## Capturas sugeridas

1. **login.png** - Pantalla de inicio de sesión
2. **register.png** - Pantalla de registro
3. **home.png** - Página principal después de iniciar sesión
4. **actors_index.png** - Listado de actores
5. **films_index.png** - Listado de películas
6. **categories_index.png** - Listado de categorías
7. **create_film.png** - Formulario de creación de película
8. **admin_panel.png** - Panel de administración

## Cómo tomar las capturas

1. Ejecuta la aplicación con `dotnet run`
2. Abre `http://localhost:5164` en tu navegador
3. Inicia sesión con uno de los usuarios predefinidos
4. Navega por las diferentes secciones
5. Toma capturas de pantalla (Windows: `Win+Shift+S`, macOS: `Cmd+Shift+4`)
6. Guarda las imágenes en esta carpeta con los nombres indicados arriba
